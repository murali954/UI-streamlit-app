# 🌾 Agricultural Commodity Forecasting & Chatbot Integration

Gain insights into future prices of various agricultural commodities and interact with an AI-powered chatbot for detailed, natural language responses.



---![WhatsApp Image 2025-06-10 at 1 27 49 PM](https://github.com/user-attachments/assets/77b33283-df53-411e-ad69-f9e78ab5fb49)


## 🚀 Features

- 📈 **Price Forecasting:** Predicts future prices using ARIMA model
- 💬 **AI Chatbot:** Ask questions like "Min price of mango" and get contextual responses using Cohere + LangChain
- 📊 **Interactive Charts:** Visualize historical and predicted price trends
- 📋 **Data Overview:** Displays region-wise commodity data
- 🌍 **Commodity Selection:** Choose specific commodity and region for targeted forecasts

---

## 🛠️ Tech Stack

- **Frontend:** Streamlit  
- **Backend Models:** ARIMA (statsmodels), BERT (via Cohere)  
- **NLP Chain:** LangChain  
- **Data Handling:** Pandas, NumPy  
- **Visualization:** Matplotlib, Seaborn  
- **Deployment:** FastAPI-ready (optional)
